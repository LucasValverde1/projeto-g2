# Projeto G2 — Vendas em E-commerce no Brasil

Este projeto desenvolve uma análise exploratória e um dashboard interativo sobre vendas em e-commerce no Brasil, utilizando Python, Pandas, Matplotlib, Seaborn, Streamlit e SQLite.

## Tema
Tema 13 — Vendas em E-commerce no Brasil

Base utilizada: `simulacao_ecommerce_brasil.csv`

Fonte dos dados: repositório Dados-Simulados-G2.

## Objetivo
Analisar o comportamento das vendas em e-commerce, observando faturamento, pedidos, ticket médio, categorias, regiões, formas de pagamento e evolução temporal.

## Tecnologias utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Streamlit
- SQLAlchemy
- SQLite
- GitHub
- GitHub Pages
- Streamlit Community Cloud

## Funcionalidades
- Filtros múltiplos no Streamlit
- KPIs dinâmicos
- Gráficos comparativos
- Análise temporal
- Correlação estatística
- Persistência em banco SQLite com SQLAlchemy
- Dashboard organizado em seções
